# Турнік-Герой — iPhone та iPad

Нативна SwiftUI-оболонка для застосунку Давида. Усередині застосунку відкривається локальна версія тренувальної програми через `WKWebView`, тому адресного рядка браузера немає, а прогрес зберігається локально на пристрої.

## Що включено

- універсальний таргет для iPhone та iPad;
- підтримка вертикальної та горизонтальної орієнтації iPad;
- локальне збереження прогресу;
- одне денне завдання на календарний день;
- окрема іконка та launch screen;
- щоденне локальне нагадування з вибором часу в розділі «Налаштування»;
- короткий звук і вібрація після завершення підходу;
- сильніший звук і success-haptic після завершення тренування.

## Запуск у Xcode

1. Відкрийте `PullUpHero.xcodeproj` у Xcode.
2. Виберіть таргет **PullUpHero** → **Signing & Capabilities**.
3. Виберіть свій **Team**.
4. Bundle Identifier уже встановлено: `com.lazarchuk.pullupheropropro`.
5. Виберіть підключений iPhone, iPad або Simulator і натисніть **Run**.

Для TestFlight: **Product → Archive → Distribute App → App Store Connect → Upload**.

## Перевірка нагадувань

Відкрийте вкладку **Налаштування**, активуйте **Нагадувати**, виберіть час і надайте iOS дозвіл на сповіщення. У Simulator вібрація не відтворюється; її потрібно перевіряти на реальному iPhone або iPad.
